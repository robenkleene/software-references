# macOS Dock

## Applications

- `⌘-click`: Reveal
- `⌥-click`: Hide the frontmost app, and switch to the clicked app
- `⌥⌘-click`: Hide other apps

## Folders

- `⌘-click`: Reveal
- `⌥⌘-click`: Open the folder in the Finder

## App Contextual Menu

### App Window Icons

- Dot: Unsaved changes
- Checkmark: Frontmost window
- Diamond: Minimized in the Dock
