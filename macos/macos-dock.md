# macOS Dock

- To reveal an app, `⌘-click` it
- To hide an app, `⌥-click` it
- To reveal an item in a folder, `⌘-click` it
- To open the contents of a folder `⌥⌘-click` it

## App Contextual Menu

### App Window Icons

- Dot: Unsaved changes
- Checkmark: Frontmost window
- Diamond: Minimized in the Dock
