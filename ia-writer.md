# iA Writer

- `⌃⌘←` / `^⌘→`: Go backward / forward
- `⌘↩` / `⌘-click`: Open link

## Format

- `⌥⌘X`: Check checkbox
- `⌘J`: Inline code
- `⇧⌘J`: Code block

## Library

- To add folder, add to `Locations` by clicking the plus button that only appears on hover (dragging to `Favorites` only works for items already added to the library)
