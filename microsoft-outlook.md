# Outlook

- `⌘1`: Mail
- `⌘2`: Calendar

## Mail

- `⌃E`: Archive
- `⌘T`: Mark as read
- `⌃1` / `⌃0`: Flag / Unflag
- `⌘↩`: Send email

## Filter

- `⌥⌘O`: Show only flagged

## Calendar

- `⌘T`: Today
- `⌥⌘←` / `⌥⌘→`: Previous / next week
- `⌃⌘1`: Day
- `⌃⌘3`: Week
- `⌃⌘4`: Month

## Troubleshooting

- To fix stuck todos, `Tools > Accounts...`, then `...` menu on account to `Reset Account`

## Workflow

- For reading email, just read the message and then either archive or flag it, i.e., don't worry about managing read states.
